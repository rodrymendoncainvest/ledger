# Ledger — Dynamic Weights

Este documento define a prioridade relativa dos blocos de análise
(growth, balance, valuation, technical, etc.)
consoante o setor ou industry.

Os pesos são qualitativos (alto / médio / baixo).
Não existem cálculos explícitos.
Servem para orientar a leitura e o tom da análise.

Se uma industry não estiver listada,
aplicar os pesos do respetivo setor.
Se nem industry nem setor existirem,
usar pesos neutros do framework base.
